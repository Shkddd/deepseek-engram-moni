# 🎯 Habit Tracker - 个人习惯追踪器

简洁高效的命令行习惯养成工具

## 功能特性

- ✅ 创建和管理个人习惯
- ✅ 每日打卡记录
- ✅ 进度统计与可视化
- ✅ 数据本地存储（JSON）
- ✅ 连续打卡天数统计

## 快速开始

```bash
python3 main.py
```

## 命令说明

| 命令 | 说明 |
|------|------|
| `add <习惯名>` | 添加新习惯 |
| `list` | 查看所有习惯 |
| `check <习惯名>` | 今日打卡 |
| `stats` | 查看统计报表 |
| `delete <习惯名>` | 删除习惯 |
| `help` | 显示帮助 |

## 技术栈

- Python 3.8+
- 数据存储：JSON
- 无外部依赖
